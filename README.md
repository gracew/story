# speakeasy

## Dev setup

### Prerequisits

- Node 10 (recommend using NVM) + Yarn
- Install firebase cli globally: `npm install -g firebase-tools`

### Install dependencies

```
cd functions
yarn
```

```
cd app
yarn
```

### Running the app locally
```
cd app
yarn start
```

## Deploying functions

```
cd functions
yarn deploy
```
